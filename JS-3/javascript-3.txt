NaN === Not a Number : برای زمانیکه برای رشته ها از عملگرهای ریاضی استفاده کنیم. یک خطا هست به این معنی که اون متغیر که روش عملگر ریاضی استفاده شدخ عدد نیست

----------------- Second ----------------
String Methods === متدهای تعریف شده برای رشته ها
1. toString: برای تبدیل متغیر با تایپ عدد به رشته
2. toFixed: اگر یک عدد اعشاری با تعداد اعشار زیاد داشته باشیم و بخواهیم سه عدد بعد از ممیز نمایش داده بشه توی برانتز عدد سه رو مینویسیم
3. Number: برای تبدیل رشته عددی استفاده میشه
4. parseInt: همون کار بالا رو میکنه با این تفاوت که اگر عدد اعشاری بود فقط صحیح اون عددو برمیگردونه
5. parseFloat: همون کاری که بالایی میکنه ولی کل عددو نمایش میده بهمون
5. backslash(\): بک اسلش اگر توی متنی داشته باشیم میتونیم ادامه متنو از خط بعد بنویسیم.اگرم بک اسلش با  -ان- انگلیسی داشته باشیم نتیجه کد ها هم از خط بعد نشون داده میشن
6. indexOf: برای پیدا کردن ایندکس یه آیتم  استفاده میشه 
7. charAt: برای اینکه ببینیم ایندکس مورد نظرمون چه کاراکتذیه
8. includes: اینکه یک کلمه یاآیتم شامل متنمون هست یا نه 
9. toUpperCase: واسه نمایش بصورت حروف کوچک
10. toLowerCase: نمایش بصورت حروف بزرک
11. split: سهس ورودی میگیره و بر اساس همون میاد و متنو به ئو قسمت تبدیل میکنه و توی یه آرایه بهمون میده
12. replace: دو مقدار میگیره مقدار دوم جای مقدار اولو توی متن میگیره
13. trim: برای پاک کردن فاصله های اول و آخر متن
14. concat: با اسن متد میشه دو یا چند متنو با هم متصل کرد و کلمات و متن هاییم میتونه توی خود متد بگیره
15. slice: برای جدا کردن یه قسمت از یک متن که دوتا مقدار میگیره از اولی شروع میکنه تا یکی قبل از عدد دومی -- ایندکس -- کاراکتزمونو وارد میکنیم
16. substr: شبیه به متد بالاییه
17. substring: از اولی شروع میکنه تا دومی هر عددی باشه به همون اندازه میره جلو و جدا میکنه
18. Template Literals(``): برای استفاده از متغیر داخل یک متن خلی استفاده داره  بک تیک و علامت دلار و آکولاد
19. length: برای بدست آموردن طول متنمون
